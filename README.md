# py-tetris
Tetris written in Python using Pygame

PROJECT SETUP
- python3 -m venv venv
- source venv/bin/activate
- sudo easy_install pip==21.0.1
- pip install -r requirements.txt

GIT SETUP
- git config --local user.name <username>
- git config --local user.email <email>
